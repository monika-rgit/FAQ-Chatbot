# FAQ Chatbot

## Overview
This project is a Python-based FAQ Chatbot that answers user questions using Natural Language Processing (NLP). It identifies the most relevant answer from a predefined set of frequently asked questions using TF-IDF Vectorization and Cosine Similarity.

## Features
- Answers frequently asked questions
- NLP-based text processing
- TF-IDF Vectorization
- Cosine Similarity matching
- Simple and easy-to-use interface

## Technologies Used
- Python
- NLTK
- Scikit-learn

## Project Structure

## How to Run

1. Install Python.
2. Install the required libraries:
   ```
   pip install nltk
   pip install scikit-learn
   ```
3. Run the program:
   ```
   python chatbot.py
   ```

## Sample Input
```
What is Python?
```
